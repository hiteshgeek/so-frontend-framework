# Project Structure

This document describes the directory layout and file organization of the SixOrbit UI Framework.

---

## Root Directory

```
so-framework/
├── src/                 # Source files (SCSS, JS)
├── dist/                # Compiled output (generated)
├── demo/                # Demo application
├── docs/                # Documentation
├── build.js             # Build script
├── package.json         # Node dependencies
└── README.md            # Project readme
```

## File Naming Conventions

### SCSS Files

| Pattern | Purpose | Example |
|---------|---------|---------|
| `_*.scss` | Partial (imported) | `_buttons.scss` |
| `*.scss` | Entry point | `sixorbit-full.scss` |

### JavaScript Files

| Pattern | Purpose | Example |
|---------|---------|---------|
| `so-*.js` | Component class | `so-modal.js` |
| `sixorbit-*.js` | Bundle entry | `sixorbit-full.js` |

### CSS Classes

| Pattern | Purpose | Example |
|---------|---------|---------|
| `.so-*` | Component | `.so-btn`, `.so-card` |
| `.so-*-*` | Modifier | `.so-btn-primary` |
| `.so-*--*` | BEM modifier | `.so-btn--lg` |

## Import Hierarchy

### SCSS

```
sixorbit-full.scss
├── @use 'abstracts'
├── @use 'base'
├── @use 'layout'
├── @use 'components'
└── @use 'utilities'
```

### JavaScript

```
sixorbit-full.js
├── import './core/so-config.js'
├── import './core/so-component.js'
├── import './components/so-*.js'
└── import './features/so-*.js'
```

## Adding a New Demo Page

1. Create page file in appropriate directory:
   ```
   demo/elements/my-component.php
   ```

2. Add to sidebar menu:
   ```json
   // demo/data/sidebar-menu.json
   { "id": "my-component", "title": "My Component", "url": "elements/my-component.php" }
   ```

3. Page template:
   ```php
   <?php
   $pageTitle = 'My Component';
   $pageDescription = 'Description of the component';

   require_once '../includes/config.php';
   require_once '../includes/header.php';
   require_once '../includes/sidebar.php';
   require_once '../includes/navbar.php';
   ?>

   <main class="so-main-content">
       <div class="so-page-header">
           <div class="so-page-header-left">
               <h1 class="so-page-title"><?= $pageTitle ?></h1>
               <p class="so-page-subtitle"><?= $pageDescription ?></p>
           </div>
       </div>
       <div class="so-page-body">
           <!-- Content here -->
       </div>
   </main>

   <?php require_once '../includes/footer.php'; ?>
   ```
