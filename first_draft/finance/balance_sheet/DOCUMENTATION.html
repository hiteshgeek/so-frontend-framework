Balance Sheet Plugin

Developer Documentation - Version 1.0.0

Overview

The Balance Sheet plugin displays a two-section balance sheet with Liabilities and Assets side by side, featuring nested groups, ledgers, sunburst charts, and fullscreen visualization capabilities.

Features

  • Two-section layout: Liabilities and Assets
  • Hierarchical nested groups and ledgers
  • Interactive sunburst charts with fullscreen view
  • Expand/collapse functionality for groups
  • Toggle debit/credit columns
  • Clickable groups and ledgers with callbacks
  • Responsive design for all devices
  • Material Design icons and styling

Basic Usage

codeJavaScript 
const balanceSheet = new BalanceSheet('container-id', {
    data: balanceSheetData,
    showDebitCredit: true,
    onGroupClick: function(group) {
        console.log('Group clicked:', group);
    },
    onLedgerClick: function(ledger) {
        console.log('Ledger clicked:', ledger);
    }
});

API Methods

loadData(data)

Load new balance sheet data

codeJavaScript 
balanceSheet.loadData(newData);

expandAll()

Expand all groups

codeJavaScript 
balanceSheet.expandAll();

collapseAll()

Collapse all groups

codeJavaScript 
balanceSheet.collapseAll();

toggleDebitCredit()

Toggle debit/credit column visibility

codeJavaScript 
balanceSheet.toggleDebitCredit();

Fullscreen Charts

The plugin includes fullscreen functionality for sunburst charts. Click the fullscreen button on any chart to view it in an enlarged modal with better visibility.

Features: Click outside modal or press ESC to close. Charts are automatically re-rendered in fullscreen for optimal display.
