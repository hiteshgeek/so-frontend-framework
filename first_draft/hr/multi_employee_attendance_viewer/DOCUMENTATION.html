Overview

The Multi Employee Attendance Viewer is a comprehensive attendance tracking plugin that displays employee attendance data across multiple days with advanced filtering, search, and visualization capabilities.

Module: HR
Version: 1.0.0
Type: Framework-agnostic JavaScript

Key Features

  • Multi-day calendar view with frozen employee column
  • Real-time statistics (present, absent, leave)
  • Advanced filtering (search, status, department, date range)
  • Attendance method indicators (biometric, mobile, manual)
  • Color-coded status badges
  • Warning indicators (late arrival, insufficient hours)
  • Mobile optimized and fully responsive

Data Format

Employee Object Structure

code JSON 
const employees = [
    {
        id: "E001",                    // Required: Unique identifier
        name: "John Doe",              // Required: Employee name
        employee_code: "EMP001",       // Required: Employee code
        department: "Sales",           // Required: Department
        designation: "Sales Manager"   // Required: Job title
    }
];

Dates Array

code JSON 
const dates = [
    "2025-01-01",  // YYYY-MM-DD format
    "2025-01-02",
    "2025-01-03"
    // ... more dates
];

Attendance Data Structure

code JSON 
const attendanceData = {
    "E001": {                          // Employee ID
        "2025-01-01": {                // Date (YYYY-MM-DD)
            status: "present",         // Required: present/absent/leave/weekend
            in_time: "09:00",          // Optional: Check-in time (HH:MM)
            out_time: "18:00",         // Optional: Check-out time (HH:MM)
            hours: 9,                  // Optional: Total hours worked
            method: "biometric"        // Optional: biometric/mobile/manual
        },
        "2025-01-02": {
            status: "leave",
            notes: "Sick leave"        // Optional: Additional notes
        }
    }
};

Status Values

Status Description Color
"present" Employee was present ● Green
"absent" Employee was absent ● Red
"leave" Employee was on leave ● Blue
"weekend" Weekend/Holiday ● Orange

Available Methods

updateFilters(filters)

Update filter criteria to show specific data.

JavaScript 
attendanceManager.updateFilters({
    search: "john",              // Search by name/code
    status: "present",           // Filter by status
    department: "Sales",         // Filter by department
    startDate: "2025-01-01",    // Date range start
    endDate: "2025-01-31"       // Date range end
});

getStatistics()

Get current attendance statistics.

JavaScript 
const stats = attendanceManager.getStatistics();
console.log(stats);
// Returns:
// {
//     totalEmployees: 50,
//     totalPresent: 45,
//     totalAbsent: 3,
//     totalLeave: 2,
//     attendanceRate: 90.0
// }

exportToCSV()

Export current attendance data to CSV file.

JavaScript 
// Export current view to CSV
attendanceManager.exportToCSV();

Reloading with Fresh Data

reload(newData)

Reload the attendance viewer with fresh data from your server or database.

JavaScript 
// Fetch fresh data from your API
const freshData = await fetch('/api/attendance').then(r => r.json());

// Reload viewer with new data
attendanceManager.reload({
    employees: freshData.employees,
    dates: freshData.dates,
    attendanceData: freshData.attendance
});
Pro Tip: The reload() method preserves filter states and view preferences, making it perfect for real-time updates or period changes.

Dynamic Data Loading & Auto-Refresh

JavaScript 
// Initialize with initial data
let attendanceManager = new AttendanceManager('attendance-container', {
    employees: initialEmployees,
    dates: initialDates,
    attendanceData: initialAttendance
});

// Function to refresh attendance data
async function refreshAttendance() {
    try {
        const response = await fetch('/api/attendance/current-month');
        const data = await response.json();

        // Reload with fresh data
        attendanceManager.reload({
            employees: data.employees,
            dates: data.dates,
            attendanceData: data.attendance
        });

        console.log('Attendance data refreshed');
    } catch (error) {
        console.error('Failed to refresh:', error);
    }
}

// Auto-refresh every 5 minutes
setInterval(refreshAttendance, 300000);

// Manual refresh button
document.getElementById('refreshBtn').addEventListener('click', refreshAttendance);
