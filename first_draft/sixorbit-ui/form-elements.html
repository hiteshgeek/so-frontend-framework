Buttons

Reusable button components with multiple variants, sizes, and states. Use .so-btn as the base class.

Button Variants
<!-- Primary - Main action -->
<button class="so-btn so-btn-primary">Primary</button>

<!-- Secondary - Alternative action -->
<button class="so-btn so-btn-secondary">Secondary</button>

<!-- Success - Positive action -->
<button class="so-btn so-btn-success">Success</button>

<!-- Danger - Destructive action -->
<button class="so-btn so-btn-danger">Danger</button>

<!-- Warning - Caution action -->
<button class="so-btn so-btn-warning">Warning</button>

<!-- Ghost - Transparent background -->
<button class="so-btn so-btn-ghost">Ghost</button>

<!-- Link - Text-only button -->
<button class="so-btn so-btn-link">Link</button>
Button Sizes
<!-- Small - Height: 32px -->
<button class="so-btn so-btn-primary so-btn-sm">Small</button>

<!-- Default/Medium - Height: 40px -->
<button class="so-btn so-btn-primary">Default</button>

<!-- Large - Height: 48px -->
<button class="so-btn so-btn-primary so-btn-lg">Large</button>
Buttons with Icons

</button>

<button class="so-btn so-btn-icon so-btn-primary so-btn-sm">
    <span class="material-icons">add</span>

Button Groups

    <button class="so-btn">Left</button>

Usage Guidelines
  • Primary: Use for the main action on a page (Save, Submit, Create, etc.)
  • Secondary: Use for alternative or less important actions (Cancel, Back, etc.)
  • Success: Use for positive confirmations (Approve, Confirm, Complete)
  • Danger: Use for destructive actions (Delete, Remove, Archive)
  • Warning: Use for caution actions (Suspend, Disable, Force Update)
  • Ghost: Use for tertiary actions with minimal visual weight
  • Link: Use for navigation or inline actions within text
  • Icon-only: Always include a title attribute for accessibility
  • Button Groups: Use for related actions or toggle options (alignment, view modes)
